Local One-Click Setup

For Windows users, clone once and run once. The launcher installs dependencies and starts backend plus frontend.

Step 1: Clone the repository.
Step 2: Move into the project folder.
Step 3: Run launcher script and open studio.
Windows CMD Recommended 
git clone https://github.com/purushothaman-natarajan/DL-Studio.git
cd DL-Studio
run_studio.bat
Browsers cannot run local CMD or PowerShell commands directly for security reasons. Copy the commands and run them in your terminal.
