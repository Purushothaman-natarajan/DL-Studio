Local One-Click Setup

For Windows users, clone and run once. The script installs dependencies and launches backend + frontend.

Windows CMD Recommended 
git clone https://github.com/purushothaman-natarajan/DL-Studio.git
cd DL-Studio
run_studio.bat
Browsers cannot execute local CMD or PowerShell commands directly for security reasons. Use copy, paste into terminal, and run.

Setup Notes

For local one-click startup on Windows use run_studio.bat or main.ps1.

Detected OS: Checking...
Target App: Backend on port 8000 and frontend on port 3000
After Launch: Open http://localhost:3000 in browser
If pip install looks stuck

Wait at least 2 to 5 minutes on first run. Large ML packages can take time. Avoid pressing Ctrl+C while install is running.

If you see "Terminate batch job (Y/N)?"

Choose N to keep setup running. Choosing Y stops the setup process immediately.

Re-run cleanly

Close old terminals, then run run_studio.bat again from the repository root folder.
